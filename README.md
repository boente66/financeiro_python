# financeiro_python# 💰 Sistema de Controle Financeiro

Sistema desktop desenvolvido em Python com PyQt para gerenciamento financeiro completo.

## 🚀 Funcionalidades

- Controle de contas
- Gestão de cartões de crédito
- Registro de receitas e despesas
- Transferências entre contas
- Importação de extratos
- Exportação (CSV, XLSX, PDF)

## 🖥️ Tecnologias

- Python
- PyQt5
- SQLite
